# 보안게시판 크롤러 (v0.1)

## 1. 목표
- 제공되어진 웹페이지 게시판에 신규 글이 추가되는지 주기적으로 확인하여 결과를 Slack 으로 알려주기

## 2. 요구사항
- 별도의 yaml 파일을 통해 모니터링할 웹페이지 게시판 url 들을 관리한다.
- yaml 파일내 url 들을 daily 로 모두 확인하여, 신규로 게시판 글이 추가되었는지 확인한다.
- 확인한 결과를 slack webhook 으로 전달한다.
- 변경사항 없으면 없다고 알려주고,
- 변경사항이 있다면 변경사항 내용을 전달한다.

## 3. 기술적 요구사항
- python 으로 개발
- uv 를 사용하여 개발
- Github Public Repository 이용
- Github Actions 를 이용하여 daily cron 작업 수행하도록 함
- 개발 과정에서 필요한 Secrets 는 보안적으로 보관하도록 개발
